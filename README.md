# Techtopia Homework 

## Prerequisites
1. Node version 20.19.5 or above

## Install instructions 

1. git pull from `https://github.com/shay-te/techtopia-homework.git`
2. open terminal 
3. cd into to project directory 
4. Install the backend `npm i`
5. Run the backend `npm run dev`
6. go into the `scr/Client` folder
7. Install the backend `npm i`
8. Run the backend `npm run dev`
9. open your braoswer and navigate tpo `http://localhost:4028/create`


## Run the backend tests
To run the tests type
`npm run test`
